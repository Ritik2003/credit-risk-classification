# Credit Risk Classification

Machine learning project to predict whether a loan applicant will default.  
Focus on **recall** due to high cost of missing positive cases.

## Features
- Class imbalance handling  
- Random Forest model  
- Evaluation using recall

## Usage
1. Put dataset in `data/credit_data.csv`  
2. Run `src/model.py`  
3. Use `requirements.txt` to install dependencies

## Technologies
Python, Pandas, Scikit-Learn
